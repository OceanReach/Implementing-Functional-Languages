则是在将函数体转为一系列的指令，函数被转为一系列指令。
比如；2+3*4变为[INum 2, INum 3, INum 4, IMult, IPuls]这样的指令序列，这样的指令序列通过相关的栈操作完成求值。
再比如：f x y = K (g x)则利用一下栈操作[Push 1, Push 1, Mkap, Pushglobal K, Mkap, Slide 3, Unwind]
